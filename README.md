# Telecom-Customer-Churn-with-Ensemble-Learning-Classifiers
This project focuses on developing an ensemble ML (RF, XgBoost, Adaboost, and LightGMB) system with hyperparameter optimization to predict customer churn in the telecommunications industry. It covers the entire data science lifecycle, from exploratory data analysis (univariate &amp; bivariate ) to model deployment on AWS EC2.
